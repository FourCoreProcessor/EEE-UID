INTRODUCTION

Wind is really a very significant method of making clean energy, but it is tricky because the wind does not blow at the same speed all the time. This makes it hard for wind turbines to always get as much power as they can. Think of it like trying to catch falling leaves - it's a lot easier if theyd are falling steadily than if they are swirling around in a crazy wind! To help with this, we use something called MPPT, which is like a smart system that helps the turbine adjust to catch the most energy, even when the wind changes. Some older ways like Perturb & Observe (P&O), Tip Speed Ratio (TSR), and Fuzzy Logic Control aren't great at handling really fast changes in wind. But there is a really cool new idea called Reinforcement Learning, where a computer program learns to adjust the turbine by trying different things and getting feedback, kind of like how you learn a video game! This helps the turbine get more power and also be gentler on the parts so they don't wear out as quickly.
